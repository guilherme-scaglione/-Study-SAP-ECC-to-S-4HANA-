# ECC Current State – Example Landscape

This document describes a typical SAP ECC system used as the source
for an SAP S/4HANA system conversion study.

## System Overview
- SAP ECC 6.0
- Unicode system
- Linux x86_64
- Database: DB2 LUW
- SAP GUI for Windows

## Key Characteristics
- Classical table-based data model
- Custom Z programs and reports
- Interfaces to external systems
- Traditional FI, MM, SD modules
