## A simplified example
# Starting at high level: Clients
 - Each client is a self-contained business units
 - Types of clients:
    [NON-PRODUCTION] developing, quality assurance
    [PRODUCTION] end-users log in here, system of record for a business, three digit number is assigned

   # Company
   - Unit where financial statements are created

   # Company codes
   - The smallest unit for which a complete self-contained set of accounts can be created.
   - Financial reports (P&L) can be run and create balance sheets for an entire company code.
   - Withing company code we have UNITS: plants (lvl below: storage location), purchasing organizations, sales organization (lvl below: distribution channels, division)
